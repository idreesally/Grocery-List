## 1.0 System Overview:
The GroceryListApp allows the user to organize items under more than one Grocery lists.
### 1.1 System Overview:
   Functionalities: 
   1. Add a new list to the existing ones 
   2. Delete all or an individual grocery lists 
   3. Rename the existing lists
   4. Add items to the lists according their types. 
   5. Specify the quantiy of the item.
   6. Check/Uncheck items in the list.
   7. Delete Item(s) from the list
## 2.0 System Summary: 
Information about system's hardware and software. 
### 2.1 System Configuration:
GroceryList App is compatible with Android 5.0 and API level 21.
## 3.0 Getting Started: 
### 3.1 Installation: 
Installation can be performed directly on an Android device. 
### 3.2 Basic User Actions: 

#### 3.2.1 Create Grocery List:

* Choose "+" from the main window.
* Name your Grocery List (in alphabetical order)
* Click "SAVE" to save your Grocery list.

#### 3.2.2 Delete All Grocery Lists:

* Choose "Reset Database" option from the main menu.
* An empty list page is seen.

#### 3.2.3 Delete an individual Grocery List:

* Select "X" icon displayed in front of the list.
* Select the list you want to delete.
* Select "Confirm" to confirm the deletion.

#### 3.2.4 Rename Grocery List:

* Select rename icon :pencil2: right next to the list.
* Type a new name on the pop up window.
* Click save.

### 3.3.0  Items menu:

####  3.3.1. Add Item and Specify the quantity.

* Select the list to which you want to add an item. 
* On the pop up window, enter the item's name and item's quantity.
* Item added to the list in alphabetical order.
* Click "SAVE".
<img src="https://github.com/qc-se-spring2018/370Spring18Team3/blob/master/GroupProject/Docs/images/AddItemAndQuantity.png" width="200" height="300" />


####  3.3.2. Add Item By Search
* Select the main menu
* Click on SearchItemActivity
* Search the items from the search bar to add to the List.
